# Document Tracker App

Полноценный веб-сервис для личного отслеживания согласования документов с постоянным хранением в PostgreSQL.

## Что внутри

- `public/index.html` — фронтенд на основе вашего последнего HTML-варианта
- `src/server.js` — backend на Express + API + подключение к PostgreSQL
- `render.yaml` — конфиг для деплоя на Render
- `package.json` — зависимости проекта

## Возможности

- хранение документов в PostgreSQL, а не в браузере;
- доступ к одним и тем же данным с ПК и телефона;
- сохранение контрагентов в отдельной таблице и автоподсказки;
- добавление, обновление и удаление документов через API;
- готовность к деплою через GitHub + Render.

## Как загрузить на GitHub

1. Создайте новый репозиторий на GitHub.
2. Загрузите в него все файлы из этой папки с сохранением структуры.
3. Убедитесь, что `package.json`, `render.yaml`, `README.md`, папки `src/` и `public/` лежат в корне репозитория.

## Как задеплоить на Render

1. В Render нажмите **New +** → **Blueprint**.
2. Подключите репозиторий GitHub.
3. Render прочитает `render.yaml` и создаст:
   - Web Service `document-tracker-app`
   - PostgreSQL `document-tracker-db`
4. Дождитесь завершения деплоя.

## Важно

- данные теперь сохраняются в базе PostgreSQL;
- они не зависят от конкретного браузера;
- сервис можно открывать с разных устройств;
- для дальнейшего шага можно добавить авторизацию.
